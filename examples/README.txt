In this folder you can find some examples of application generated with the MobileCodeGenerator and then completed.
For each application there are two subfolder:
- generated: it contains the source code generated with the MobileCodeGenerator with the .xmi model which is contained in the model folder of the MobileCodeGenerator (it.polimi.mobilecodegenerator --> src --> model). This source code is generated in the src-gen folder of the MobileCodeGenerator.
- completed: it contains the source code modified and completed with Eclipse ADT for the Android version of the application and Xcode for the iOS version of the application.
